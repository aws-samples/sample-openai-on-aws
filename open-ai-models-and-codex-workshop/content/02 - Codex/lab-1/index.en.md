Configure Codex for this workshop to run GPT models on Amazon Bedrock. Lab 1 sets up the **Codex App** first, then the **CLI**, and offers the IDE extension for attendees who prefer it.

## 1. What you'll learn

In this lab, you will:

- Clone the `codex-advanced-patterns` branch of `aws-samples/sample-openai-on-aws`
- Install the Codex App and point it at Bedrock with the built-in `amazon-bedrock` provider
- Open `sample-openai-on-aws/bedrock-chat` in the App and run a first codebase exploration prompt
- Configure the Codex CLI for Bedrock, reusing your AWS credentials from Getting Started
- Optionally connect the Codex IDE extension
- Plan Bedrock service quotas and cost monitoring for a team rollout
- (Optional) Choose authentication patterns for your team deployment
