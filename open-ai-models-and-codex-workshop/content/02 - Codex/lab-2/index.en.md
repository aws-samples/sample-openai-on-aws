In this lab, you will learn advanced Codex techniques to enhance your development workflow and enable effective team collaboration. The recommended path uses the Codex App for exploration, planning, diffs, and validation, with IDE and CLI examples included where they are useful. You'll establish team standards, master goals and plan mode, scale workflows with skills, and manage large projects efficiently.

:::alert{type="info" header="Start from the cloned project"}
These exercises assume you are working in `~/workshop/sample-openai-on-aws/bedrock-chat`, cloned from the `codex-advanced-patterns` branch during Lab 1.
:::

:::alert{type="info" header="Additional resources"}
For comprehensive workflow patterns, see the [Codex CLI documentation](https://developers.openai.com/codex).
:::

## 1. What you'll learn

In this lab, you will:

- Apply the EPCC workflow (Explore-Plan-Code-Commit) for structured development
- Use the Codex App for practical workflows like codebase understanding, feature work, review, image-driven UI tasks, and test-driven fixes
- Establish team standards with AGENTS.md files and custom slash commands
- Use goals and plan mode for structured, spec-driven development
- Scale workflows with skills and memories for reusable domain expertise
- Manage large projects with session management, checkpoints, subagents, and custom agents
- (Optional) Explore session productivity tips: side conversations, image inputs, and vim mode
- (Optional) Use git and parallel workflows: subagents, fork conversations, and concurrent work
- (Optional) Extend Codex with integrations: MCP servers for external tools and IDE integration
