In this lab, you'll learn how to invoke GPT-5.5 using the OpenAI SDK. The SDK is the foundation for all subsequent labs. State, function calling, structured outputs, vision, and agents all build on it.

After the basic SDK call, a short state-management exercise shows when to use `previous_response_id` versus manual history. An optional curl module covers raw HTTP requests for those who want to understand what happens under the hood, and the structured outputs exercise shows how to get model responses back as JSON or typed Python objects.
